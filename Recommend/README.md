### 1. LICEcap

免费的 `Gif` 屏幕录像软件。

使用方法：录制目标可以通过拖动软件窗口随意调整大小，然后点击右下角录制按钮，选择好保存路径，其他功能按照需要选择，显示标题名暂不支持中文，点击保存，倒数 3 秒后开始录制，结束时点击停止按钮即可。

[官网地址](https://www.cockos.com/licecap/)
